# Virtual Drawing System

## Description
A real-time virtual drawing system using hand gesture recognition with MediaPipe and OpenCV, allowing users to draw, erase, and select colors interactively.

## Features
- Draw using finger gestures
- Erase using hand gestures
- Select colors using gesture-based palette
- Real-time hand tracking

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## Controls
- Index finger → Draw  
- Open palm → Erase  
- Two fingers → Select color  

## Note
The required model file will be automatically downloaded when you run the program.
